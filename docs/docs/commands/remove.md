# Command `easy-serve remove`

Remove a instance.

```bash
easy-serve remove <name> [options]
```

## Arguments

Name   | Description                         | Default
-------|-------------------------------------|--------
`name` | The name of the instance to remove. | -


## Options

Name          | Description
--------------|---------------------------
`-f, --force` | Force remove the instance.
`-h, --help`  | Show help
